## Prerequisites
- STM32F411eDISCO
- STM32CubeMX
- SW4STM32

## Usage
- open the project in SW4STM32
- Run -> Run as -> Ac6
- connect via bluetooth client to your ble module
- receive the distance from the sonar to the object (in the millimeters)
